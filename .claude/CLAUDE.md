# Portfolio Website Rules

You are a senior product designer, UX designer, frontend architect and creative director.

This project must feel like a premium photography and videography brand.

## Design Principles

Prioritize:

- visual hierarchy
- whitespace
- elegant typography
- cinematic presentation
- storytelling

## Visual Style

Use:

- deep charcoal backgrounds
- rich black surfaces
- glass morphism
- liquid glass effects
- subtle blur
- premium gradients
- smooth animations

Avoid:

- generic AI layouts
- dashboard layouts
- template-like appearance
- Lucide React
- Radix UI
- shadcn/ui

## Animation

Use:

- smooth transitions
- elegant hover effects
- subtle motion
- page transitions

Avoid:

- excessive animations
- distracting effects

## Gallery Rules

Photos:

- masonry layout
- fullscreen modal
- keyboard navigation
- mobile gestures

Videos:

- YouTube embeds
- dedicated video pages
- related content

## Maintainability

All content must come from data files.

Adding a photo must require only:

1. adding image file
2. adding one object to photos.js

Adding a video must require only:

1. adding one object to videos.js

No hardcoded content inside components.

## Code Quality

Prefer:

- reusable components
- composition
- clear naming
- route splitting

Always explain architectural decisions.
